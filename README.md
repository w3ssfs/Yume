# 🌸 Yume Anime

Um catálogo de animes moderno desenvolvido com **React**, inspirado pela palavra japonesa **夢 (Yume)**, que significa **"sonho"**.

🔗 Acesse o projeto: https://yumeanime.netlify.app/

---

## 📖 Sobre o Projeto

O **Yume Anime** é um projeto pessoal criado com o objetivo de explorar tecnologias modernas de desenvolvimento web enquanto oferece uma experiência elegante para fãs de anime.

A plataforma permite descobrir animes em exibição, lançamentos, títulos populares e animes da temporada atual, além de visualizar informações detalhadas, trailers e criar uma coleção personalizada de favoritos.

O projeto foi desenvolvido como parte do meu portfólio e aprendizado contínuo em desenvolvimento Front-End.

---

## ✨ Funcionalidades

* 🎬 Exploração de animes da temporada atual
* 🔥 Animes populares e em destaque
* ⭐ Animes mais bem avaliados
* 📅 Próximos lançamentos
* 🔍 Sistema de busca de animes
* ❤️ Lista de favoritos
* 📌 Lista "Assistir Depois"
* 🎥 Exibição de trailers
* 📖 Página detalhada de cada anime
* 🔐 Login com Google
* ☁️ Sincronização de dados em tempo real
* 📱 Interface responsiva
* 🎨 Animações fluidas com Framer Motion

---

## 🚀 Tecnologias Utilizadas

### Front-End

* React
* React Router
* Framer Motion
* CSS3

### Backend & Serviços

* Firebase Authentication
* Firebase Firestore

### API

* Jikan API (MyAnimeList)

---

## 📊 Fonte dos Dados

Todas as informações sobre animes — incluindo:

* Sinopses
* Capas
* Notas
* Gêneros
* Episódios
* Trailers

são fornecidas pela **Jikan API**, uma API REST gratuita baseada nos dados do MyAnimeList.

Este projeto não é afiliado ao MyAnimeList ou ao Jikan.
